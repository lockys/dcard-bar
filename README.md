dcard-card-bar 狄卡卡霸
==
A OS X menubar helps you check what card of Dcard you get today!
![](./media/shot.png)

# Build

```sh
$ git clone https://github.com/lockys/dcard-card-bar.git dcard-card-bar && cd dcard-card-bar
$ npm run build
```

# Develop

**Install dependencies**
```sh
$ git clone https://github.com/lockys/dcard-card-bar.git dcard-card-bar && cd dcard-card-bar
$ npm install
```

**Run the bar!**
```sh
$ npm start
```
